Unordered Lists

Identify an unordered list, mark it up as a ul element. The opening <ul> tag
goes before the first list item, and the closing tag </ul> goes after the last
item. Then, each item in the list gets marked up as a list item (li) by enclosing
it in opening and closing li tags, as shown in this example. Notice that there
are no bullets in the source document. They are added automatically by the
browser. For example:

  • Fruits
  • Mango
  • Apple
  • Pineapple
  • Vegetables
  • Cabbage
  • Lettuce
  • Carrot
